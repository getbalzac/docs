## Simple & Powerful

Write content in Markdown, style with Handlebars templates, and deploy anywhere.

```toml
# balzac.toml
[global]
site_name = "My Site"
```

```handlebars
<!-- pages/index.hbs -->
<!DOCTYPE html>
<html>
  <body>
    <h1>{{site_name}}</h1>
    <p>Hello, World!</p>
  </body>
</html>
```

```bash
balzac build
```
